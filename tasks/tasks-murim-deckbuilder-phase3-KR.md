# 작업 목록: 강호무적 - Phase 3 콘텐츠 확장

## 프로젝트 개요

이 작업 목록은 무협 덱 빌딩 로그라이크 게임 "강호무적"의 **Phase 3: 콘텐츠 확장** 개발을 다룹니다. 목표는 3-4개월 내에 전체 5개 지역 및 시스템을 완성하는 것입니다.

**Phase 3 성공 기준:**
- 5개 지역 완전 구현 (각각 고유한 테마와 보스)
- 카드 120장으로 확장 (다양한 무기술 계열)
- 유물 60개로 확장 (등급별 분포)
- 내공 경지 시스템 완성 (5단계)
- 무기술 경지 시스템 완성 (6개 무기술 × 5단계)
- 이벤트 30개 이상
- 5개 시작 분파 (화산파, 소림사, 개방, 당문, 혈마신교)
- 완전한 메타 진행 시스템 (업적, 해금)
- 업적 시스템 (30개)
- 게임 전체 클리어 가능
- 외부 테스터 80% 이상 긍정 반응

**전제 조건:** Phase 2 수직 슬라이스 완료

**참고:** 상세 요구사항은 `prd-murim-deckbuilder-roguelike-KR.md`를 참조하세요.

## Phase 3 성공 지표

완료 시 다음 기준을 충족해야 합니다:

✅ **기능 완성도:**
- 5개 지역 완주 가능 (시작 → 5지역 → 최종 보스 → 승리/패배)
- 120장의 밸런스 잡힌 카드
- 60개의 기능적 유물
- 내공 경지 5단계 + 무기술 경지 6계열 × 5단계 완성
- 5개 분파 선택 가능
- 메타 진행 및 업적 시스템 완전 작동
- 무림지(도감) 수집 기능

✅ **품질:**
- 치명적 버그 0개
- 외부 테스터 긍정 반응 (80% 이상 재미있다고 평가)
- 평균 1회 클리어 플레이 타임 2-3시간
- 승률 20-30% (적절한 난이도)
- 분파별 승률 차이 10% 이내

✅ **기술:**
- 60 FPS 유지
- 세이브/로드 안정적 작동
- 모든 시스템 통합 및 상호작용 정상 작동
- 메모리 누수 없음
