# PRD: 강호무적 - 무협 덱 빌딩 로그라이크

## 1. 개요 (Introduction/Overview)

### 1.1 프로젝트 개요
**강호무적**은 무협 세계관을 배경으로 한 덱 빌딩 로그라이크 게임입니다. 플레이어는 이름 없는 낭인으로 시작하여, 매번 새롭게 생성되는 강호를 유랑하며 다양한 무공 카드를 수집하고, 자신만의 무공 덱을 구축합니다. 내공과 무기술이라는 두 가지 성장 축을 통해 경지를 돌파하며, 최종적으로 강호의 패자가 되는 것이 목표입니다.

### 1.2 해결하고자 하는 문제
- **전략적 깊이**: 단순 반복이 아닌, 매 플레이마다 다른 전략적 선택을 요구하는 덱 빌딩 경험 제공
- **무협 몰입감**: 무협 소설의 성장 서사와 강호 유랑의 낭만을 게임으로 구현
- **점진적 성장**: 로그라이크의 도전성과 메타 진행의 성취감을 균형있게 제공

### 1.3 타겟 플랫폼
- **주 플랫폼**: PC (Windows/Mac/Linux)
- **게임 엔진**: Unity 또는 Godot (개발팀 선택)
- **아트 스타일**: 2D 일러스트 + 간단한 애니메이션

## 2. 목표 (Goals)

### 2.1 비즈니스 목표
1. 덱 빌딩 로그라이크 장르의 무협 테마 시장 진입
2. Steam을 통한 인디 게임 출시 및 수익 창출
3. 긍정적 사용자 리뷰 80% 이상 달성

### 2.2 제품 목표
1. 플레이어가 100회 이상 반복 플레이할 수 있는 콘텐츠 깊이 제공
2. 1회차 클리어까지 2-3시간 소요되는 적절한 게임 길이 구현
3. 20종 이상의 무공 계열과 100장 이상의 무공 카드 제공
4. 내공/무기술 이중 성장 시스템을 통한 차별화된 빌드 다양성 제공

### 2.3 사용자 경험 목표
1. 직관적인 UI/UX로 장르 초보자도 30분 내 게임 메커니즘 이해
2. 무협 세계관 몰입을 위한 적절한 용어 및 연출 사용
3. 매 런마다 새로운 선택과 발견의 재미 제공

## 3. 사용자 스토리 (User Stories)

### 3.1 핵심 사용자 스토리

**US-001: 게임 시작 및 낭인 생성**
- **As a** 플레이어
- **I want to** 새로운 게임을 시작하고 나만의 낭인을 생성하고 싶다
- **So that** 강호 유랑을 시작할 수 있다
- **Acceptance Criteria**:
  - 게임 시작 시 기본 시작 카드 덱(10-15장)을 받는다
  - 선택 가능한 3-5개의 시작 무기술 분파 중 하나를 선택한다
  - 낭인의 기본 체력, 내공 수치가 설정된다

**US-002: 강호 맵 탐험**
- **As a** 플레이어
- **I want to** 절차적으로 생성된 강호 맵을 탐험하고 싶다
- **So that** 다양한 이벤트와 전투를 경험할 수 있다
- **Acceptance Criteria**:
  - 맵은 노드 기반 경로 선택 방식으로 구성된다 (Slay the Spire 스타일)
  - 노드 타입: 전투, 정예 전투, 휴식, 상점, 이벤트, 보스
  - 각 지역(장)은 12-18개의 노드로 구성되며, 3-5개 지역으로 게임이 구성된다

**US-003: 턴제 무공 전투**
- **As a** 플레이어
- **I want to** 수집한 무공 카드를 사용하여 적과 턴제 전투를 하고 싶다
- **So that** 전략적으로 적을 물리치고 보상을 얻을 수 있다
- **Acceptance Criteria**:
  - 매 턴 시작 시 정해진 수만큼 카드를 뽑는다 (기본 5장)
  - 내공(기력)을 소모하여 카드를 사용한다
  - 카드 타입: 초식(공격), 신법(방어/회피), 기공(버프/디버프), 비기(특수 효과)
  - 턴 종료 시 남은 카드는 버려지고 내공이 회복된다

**US-004: 무공 덱 구축**
- **As a** 플레이어
- **I want to** 전투 후 새로운 무공 카드를 획득하고 덱을 강화하고 싶다
- **So that** 점점 더 강력한 조합을 만들 수 있다
- **Acceptance Criteria**:
  - 전투 승리 후 3장의 무공 카드 중 1장을 선택하여 덱에 추가
  - 휴식/이벤트에서 카드를 제거하거나 업그레이드할 수 있다
  - 덱은 최소 20장, 최대 50장으로 제한된다

**US-005: 내공 경지 돌파**
- **As a** 플레이어
- **I want to** 내공 수련을 통해 경지를 돌파하고 싶다
- **So that** 더 강력한 내공 기반 능력을 얻을 수 있다
- **Acceptance Criteria**:
  - 내공 경지: 후천 → 선천 → 화경 → 종사 → 천인합일 (5단계)
  - 경지 돌파 시 영구 효과 획득 (예: 최대 내공 증가, 카드 뽑기 수 증가)
  - 경지는 특정 조건 달성 시 돌파 가능 (예: 내공 카드 30회 사용)

**US-006: 무기술 경지 돌파**
- **As a** 플레이어
- **I want to** 무기술 단련을 통해 경지를 돌파하고 싶다
- **So that** 무기별 특화 능력과 비기를 해금할 수 있다
- **Acceptance Criteria**:
  - 무기술 경지: 입문 → 소성 → 대성 → 화경 → 검심통명/도의 (5단계)
  - 무기 종류: 검(劍), 도(刀), 창(槍), 장(掌), 권(拳), 기문병기
  - 경지 돌파 시 해당 무기 카드 강화 및 전용 비기 해금

**US-007: 메타 진행 및 영구 업그레이드**
- **As a** 플레이어
- **I want to** 사망 후에도 일부 진행도를 유지하고 싶다
- **So that** 점진적으로 성장하며 더 깊은 콘텐츠를 즐길 수 있다
- **Acceptance Criteria**:
  - 사망 시 "무공 정수"를 획득하여 영구 업그레이드에 사용
  - 영구 업그레이드: 시작 체력 증가, 시작 골드 증가, 새로운 카드 해금 등
  - 특정 업적 달성 시 새로운 시작 분파 또는 유물 해금

**US-008: 보스 전투 및 클리어**
- **As a** 플레이어
- **I want to** 각 지역의 보스와 최종 보스를 물리치고 싶다
- **So that** 강호의 패자가 되는 성취감을 느낄 수 있다
- **Acceptance Criteria**:
  - 각 지역 마지막에 고유한 패턴을 가진 보스 등장
  - 최종 보스 격파 시 게임 클리어 및 엔딩
  - 클리어 후 더 높은 난이도 해금 (고수, 초고수, 절정고수 모드)

## 4. 기능 요구사항 (Functional Requirements)

### 4.1 게임 시작 및 메인 메뉴

**FR-001**: 메인 메뉴 구성
- 시스템은 다음 메뉴 항목을 제공해야 한다:
  - 새 게임 시작
  - 이어하기 (세이브 파일이 있는 경우)
  - 무공비전 (메타 업그레이드)
  - 무림지 (업적 및 도감)
  - 설정
  - 게임 종료

**FR-002**: 시작 분파 선택
- 플레이어는 게임 시작 시 다음 중 하나의 시작 분파를 선택해야 한다:
  - 화산파 (검술 특화)
  - 소림사 (권법 특화)
  - 개방 (장법 특화)
  - 당문 (암기 및 독 특화)
  - 혈마신교 (마공 특화)
- 각 분파는 고유한 시작 덱과 초기 유물을 제공한다

**FR-003**: 시작 덱 구성
- 각 분파의 시작 덱은 12장으로 구성된다:
  - 기본 공격 카드 × 5
  - 기본 방어 카드 × 4
  - 기본 내공 회복 카드 × 2
  - 분파 특화 카드 × 1

### 4.2 강호 맵 시스템

**FR-004**: 절차적 맵 생성
- 시스템은 매 게임 시작 시 랜덤한 노드 기반 맵을 생성해야 한다
- 맵은 3-5개의 지역(장)으로 구성된다
- 각 지역은 다음 레이어 구조를 가진다:
  - 시작 노드 (1개)
  - 중간 레이어 (10-15개 노드, 3-5개 레이어)
  - 보스 노드 (1개)

**FR-005**: 노드 타입 및 분포
- 전투 노드 (50-60%): 일반 적과의 전투
- 정예 전투 노드 (10-15%): 강력한 적, 더 좋은 보상
- 휴식 노드 (10-15%): 체력 회복 또는 카드 업그레이드
- 상점 노드 (5-10%): 카드/유물 구매, 카드 제거
- 이벤트 노드 (10-15%): 랜덤 이벤트 발생
- 보물 노드 (5%): 유물 획득
- 보스 노드 (각 지역 마지막)

**FR-006**: 노드 간 이동
- 플레이어는 현재 노드에서 연결된 다음 레이어의 노드 중 하나를 선택할 수 있다
- 이동한 노드는 방문 처리되며, 뒤로 돌아갈 수 없다
- 보스 노드에 도달하면 자동으로 보스 전투 시작

### 4.3 전투 시스템

**FR-007**: 전투 시작 및 초기화
- 전투 시작 시 다음이 초기화된다:
  - 플레이어 내공: 3 (기본값, 유물/경지에 따라 변동)
  - 덱 셔플 후 5장 드로우 (기본값)
  - 적 배치 및 의도 표시

**FR-008**: 턴 진행 순서
1. **턴 시작 단계**:
   - 내공 최대치로 회복
   - 시작 단계 효과 발동 (버프/디버프)
   - 카드 드로우 (기본 5장, 손에 10장 제한)
2. **플레이어 행동 단계**:
   - 카드 사용 (내공 소모)
   - 카드 효과 즉시 적용
   - 원하는 만큼 카드 사용 가능
3. **턴 종료 선택**:
   - 플레이어가 "턴 종료" 버튼 클릭
4. **적 행동 단계**:
   - 적의 의도 실행 (공격, 버프, 디버프 등)
5. **턴 종료 단계**:
   - 손에 남은 카드 모두 버리기 더미로 이동
   - 종료 단계 효과 발동
   - 다음 턴 시작

**FR-009**: 카드 타입 및 효과
카드는 다음 4가지 타입으로 분류된다:

1. **초식 (공격 카드)**:
   - 적에게 직접 피해 부여
   - 예시: "일검 (1내공): 적에게 6 피해"

2. **신법 (방어 카드)**:
   - 방어도 획득 (턴 종료 시 소멸)
   - 예시: "철포삼 (1내공): 방어도 5 획득"

3. **기공 (강화 카드)**:
   - 버프/디버프 부여
   - 예시: "내공 운기 (1내공): 다음 공격 카드 피해 +3"

4. **비기 (특수 카드)**:
   - 고유한 특수 효과
   - 예시: "검기방출 (2내공): 모든 적에게 10 피해, 소진(사용 시 제거)"

**FR-010**: 카드 속성
각 카드는 다음 속성을 가진다:
- **이름**: 카드의 무공 명칭
- **비용**: 사용에 필요한 내공
- **타입**: 초식/신법/기공/비기
- **계열**: 검술/도법/창술/장법/권법/내공/기타
- **등급**: 평범(회색), 고급(파란색), 진귀(보라색), 전설(금색)
- **효과**: 카드 사용 시 발동되는 효과
- **업그레이드 여부**: 업그레이드 시 효과 강화

**FR-011**: 덱 시스템
- **뽑기 더미**: 아직 뽑지 않은 카드들
- **손패**: 현재 사용 가능한 카드 (최대 10장)
- **버리기 더미**: 사용했거나 버린 카드들
- **소진 더미**: 전투 중 제거된 카드 (전투 종료 후 복구)
- 뽑기 더미가 비면 버리기 더미를 셔플하여 새 뽑기 더미 생성

**FR-012**: 적 AI 및 의도 시스템
- 각 적은 매 턴 시작 시 다음 행동 의도를 표시해야 한다:
  - 공격 (칼 아이콘 + 피해량)
  - 방어 (방패 아이콘 + 방어도)
  - 버프 (위쪽 화살표)
  - 디버프 (아래쪽 화살표)
  - 알 수 없음 (물음표)
- 적은 정해진 패턴 또는 확률적 선택으로 행동한다

**FR-013**: 전투 승리 조건
- 모든 적의 체력이 0이 되면 전투 승리
- 전투 승리 시 다음 보상 중 선택:
  - 무공 카드 보상: 3장 중 1장 선택하여 덱에 추가 (또는 건너뛰기)
  - 골드 획득: 40-60 (랜덤)
  - 유물 획득 (정예 전투, 10-30% 확률)

**FR-014**: 전투 패배 조건
- 플레이어 체력이 0 이하가 되면 사망
- 사망 시 현재 런 종료, 메타 화폐 "무공 정수" 획득
- 메인 메뉴로 돌아감

### 4.4 내공 시스템

**FR-015**: 내공 수치
- **최대 내공**: 매 턴 시작 시 사용 가능한 내공량 (기본 3)
- **현재 내공**: 턴 중 남은 내공
- 특정 유물/카드로 최대 내공 영구 증가 가능

**FR-016**: 내공 경지 시스템
내공 경지는 5단계로 구성된다:

| 경지 | 달성 조건 | 효과 |
|------|----------|------|
| 후천 | 시작 단계 | - |
| 선천 | 내공 카드 20회 사용 | 최대 내공 +1 |
| 화경 | 한 전투에서 내공 50 이상 소모 | 턴 시작 시 카드 1장 추가 드로우 |
| 종사 | 누적 내공 500 소모 | 최대 내공 +1, 내공 카드 피해 +50% |
| 천인합일 | 내공 비기 10종 사용 | 내공 비기 사용 시 내공 1 환원 |

**FR-017**: 경지 돌파 UI
- 경지 돌파 조건 달성 시 전투 종료 후 돌파 이벤트 발생
- 플레이어는 돌파 시 효과를 확인하고 수락해야 한다
- 경지는 되돌릴 수 없으며, 런 내에서 영구 적용된다

### 4.5 무기술 시스템

**FR-018**: 무기술 종류
게임은 6가지 무기술 계열을 제공한다:

1. **검술 (劍)**:
   - 특징: 균형잡힌 공격/방어, 연속 공격 특화
   - 대표 카드: 삼재검법, 무영검, 화산일검

2. **도법 (刀)**:
   - 특징: 강력한 단일 피해, 체력 소모 계열
   - 대표 카드: 쾌도난마, 혈도, 패왕도

3. **창술 (槍)**:
   - 특징: 중거리 공격, 관통/디버프
   - 대표 카드: 회마창, 용창파혼, 백전창법

4. **장법 (掌)**:
   - 특징: 내공 연계, 버프/디버프
   - 대표 카드: 항룡십팔장, 천산육양장, 소요유

5. **권법 (拳)**:
   - 특징: 다수 공격, 누적 피해
   - 대표 카드: 나한권, 철사권, 대력금강권

6. **기문병기 (奇門)**:
   - 특징: 특수 효과, 암기/독/기괴한 무기
   - 대표 카드: 암기난무, 독침, 혼천금륜

**FR-019**: 무기술 경지 시스템
각 무기술은 독립적인 경지를 가진다:

| 경지 | 달성 조건 예시 | 효과 예시 (검술 기준) |
|------|---------------|---------------------|
| 입문 | 시작 단계 | - |
| 소성 | 검술 카드 15장 보유 | 검술 카드 피해 +2 |
| 대성 | 검술 카드로 300 피해 누적 | 검술 카드 사용 시 방어도 2 획득 |
| 화경 | 한 전투에서 검술 카드만 사용하여 승리 | 검술 카드 비용 -1 |
| 검심통명 | 검술 비기 5종 사용 | 전설 검술 비기 "독고구검" 해금 |

**FR-020**: 복수 무기술 빌드
- 플레이어는 한 런 내에서 여러 무기술을 동시에 발전시킬 수 있다
- 각 무기술 경지는 독립적으로 계산된다
- 혼합 빌드 시너지를 위한 카드 제공 (예: "검도쌍수")

### 4.6 유물 시스템

**FR-021**: 유물 획득
- 유물은 영구 효과를 제공하는 아이템이다
- 획득 경로:
  - 정예 전투 승리 (30% 확률)
  - 보물 노드 (100%)
  - 상점 구매
  - 특정 이벤트
- 한 런에서 최대 20개까지 보유 가능

**FR-022**: 유물 등급 및 예시
유물은 다음 등급으로 분류된다:

**일반 유물 (회색)**:
- 낡은 검집: 전투 시작 시 방어도 3 획득
- 기혈환: 휴식 시 체력 회복량 +10

**고급 유물 (파란색)**:
- 백년인삼: 최대 체력 +20
- 천년빙백단: 턴 시작 시 카드 1장 추가 드로우

**진귀 유물 (보라색)**:
- 구양신공 서책: 내공 카드 피해 +100%
- 용린검: 검술 카드 사용 시 20% 확률로 내공 환원

**전설 유물 (금색)**:
- 소요신공: 매 턴 손에 "소요유" 카드 1장 추가 생성
- 무림맹주 영패: 상점 가격 50% 할인, 골드 획득 2배

**FR-023**: 유물 조합 시너지
- 특정 유물 조합 시 숨겨진 효과 발동
- 예시: "음양옥" + "태극진경" = 턴 종료 시 체력 3 회복

### 4.7 상점 시스템

**FR-024**: 상점 노드
- 상점 노드 진입 시 다음 옵션 제공:
  - 무공 카드 구매 (5-7장, 각 40-100 골드)
  - 유물 구매 (2-3개, 각 100-250 골드)
  - 카드 제거 (50 골드, 점점 증가: 75, 100, 150...)
  - 카드 업그레이드 (50 골드)
  - 체력 회복 물약 (30% 회복, 50 골드)

**FR-025**: 가격 책정
- 카드 가격은 등급에 따라 결정:
  - 평범: 40-60 골드
  - 고급: 60-80 골드
  - 진귀: 80-100 골드
  - 전설: 150-200 골드
- 유물 가격은 등급에 따라:
  - 일반: 80-120 골드
  - 고급: 120-180 골드
  - 진귀: 180-250 골드
  - 전설: 350-500 골드

### 4.8 이벤트 시스템

**FR-026**: 이벤트 노드
- 이벤트 노드 진입 시 랜덤한 강호 이벤트 발생
- 각 이벤트는 2-4개의 선택지를 제공한다
- 선택에 따라 다양한 결과 발생 (보상, 페널티, 추가 선택)

**FR-027**: 이벤트 예시

**이벤트: "노검객의 제안"**
- 설명: "노검객이 당신에게 검법을 전수하겠다고 제안합니다."
- 선택지:
  - A. 받아들인다 → 검술 카드 1장 획득, 최대 체력 -5
  - B. 거절한다 → 골드 50 획득
  - C. 검을 겨룬다 (검술 경지 소성 이상) → 전설 검술 카드 획득

**이벤트: "강호의 분쟁"**
- 설명: "두 세력이 싸우고 있습니다."
- 선택지:
  - A. 정파를 돕는다 → 전투 발생, 승리 시 유물 획득
  - B. 사파를 돕는다 → 전투 발생, 승리 시 마공 카드 획득
  - C. 모른 척 지나간다 → 골드 20 손실

**FR-028**: 이벤트 조건부 선택지
- 특정 조건 달성 시에만 표시되는 선택지
- 조건 예시:
  - 경지 단계 (내공 화경 이상)
  - 특정 유물 보유
  - 특정 카드 보유
  - 최대 체력 기준 (50% 이하 등)

### 4.9 휴식 시스템

**FR-029**: 휴식 노드
휴식 노드에서 다음 중 하나 선택:
- **수면**: 최대 체력의 30% 회복
- **수련**: 카드 1장 업그레이드
- **타파심마** (최대 체력 50% 이상 시): 카드 2장 업그레이드, 최대 체력 10% 손실

**FR-030**: 카드 업그레이드
- 업그레이드된 카드는 다음 중 하나 이상이 강화된다:
  - 피해량/방어도 증가
  - 비용 감소
  - 추가 효과 부여
- 카드는 최대 1회만 업그레이드 가능
- 업그레이드된 카드는 이름 옆에 "+" 표시

### 4.10 보스 전투 시스템

**FR-031**: 보스 구성
게임은 총 5개 지역, 각 지역마다 고유한 보스를 배치:

**1장: 강남 - 보스 "혈랑도객"**
- 체력: 250
- 패턴: 도법 위주의 강력한 단일 공격, 분노 시스템

**2장: 중원 - 보스 "철장문주"**
- 체력: 300
- 패턴: 장법 + 소환 (부하 소환), 반격 버프

**3장: 서역 - 보스 "마교 호법"**
- 체력: 350
- 패턴: 마공 + 디버프, 생명력 흡수

**4장: 북해 - 보스 "빙궁장로"**
- 체력: 400
- 패턴: 빙속성 공격, 둔화/빙결 디버프

**5장: 천하 - 최종 보스 "천하제일인"**
- 체력: 600
- 페이즈 2단계
- 패턴: 모든 무공 구사, 다양한 비기

**FR-032**: 보스 전용 메커니즘
- 각 보스는 일반 적과 다른 특수 메커니즘 보유:
  - 다단계 페이즈
  - 특수 버프/디버프
  - 소환 능력
  - 카운터 공격
- 보스 격파 시 특별 보상:
  - 진귀/전설 유물 (100%)
  - 진귀/전설 카드 3장 중 1장 선택
  - 골드 200-300

### 4.11 메타 진행 시스템

**FR-033**: 무공 정수 획득
- 사망 또는 클리어 시 "무공 정수" 획득
- 획득량 계산:
  - 기본: 도달한 지역 × 10
  - 처치한 적 수 × 2
  - 수집한 카드 수 × 1
  - 획득한 유물 수 × 5
  - 보스 격파 × 20
  - 클리어 보너스: 100

**FR-034**: 무공비전 (영구 업그레이드)
메인 메뉴 "무공비전"에서 무공 정수 소모하여 영구 업그레이드:

**기본 강화**:
- 시작 체력 증가 (+5, +10, +15...) - 10/20/30 정수
- 시작 골드 증가 (+30, +60, +100) - 15/30/50 정수
- 최대 내공 +1 (1회만) - 100 정수

**카드 해금**:
- 각 분파의 상위 카드 해금 - 20-50 정수
- 전설 카드 해금 - 100 정수

**분파 해금**:
- 새로운 시작 분파 해금 - 50-80 정수
- 특수 유물 해금 - 60 정수

**난이도 해금**:
- 고수 모드 해금 (적 50% 강화, 보상 1.5배) - 30 정수
- 초고수 모드 해금 (적 100% 강화, 보상 2배) - 60 정수
- 절정고수 모드 해금 (적 150% 강화, 보상 3배) - 100 정수

**FR-035**: 업적 시스템
- 특정 조건 달성 시 업적 해금
- 업적 보상: 무공 정수, 특수 유물 해금, 칭호
- 업적 예시:
  - "검성": 검술만으로 게임 클리어
  - "무결": 피해를 받지 않고 보스 격파
  - "수집가": 100종의 카드 획득
  - "부자": 한 런에서 골드 1000 이상 보유

**FR-036**: 무림지 (도감)
- 수집한 모든 카드/유물/적/보스 정보 기록
- 각 항목의 상세 설명 및 획득 방법 표시
- 완성도 표시 (카드 20/120, 유물 15/60 등)

### 4.12 세이브/로드 시스템

**FR-037**: 자동 저장
- 시스템은 다음 시점에 자동 저장:
  - 노드 이동 후
  - 전투 승리 후
  - 이벤트 선택 후
  - 상점 거래 후

**FR-038**: 세이브 슬롯
- 각 런은 독립적인 세이브 슬롯 사용
- 동시에 최대 3개의 런 진행 가능
- 세이브 슬롯 정보 표시:
  - 낭인 이름
  - 현재 지역
  - 플레이 시간
  - 분파

**FR-039**: 포기 및 삭제
- 플레이어는 언제든지 현재 런을 포기할 수 있다
- 포기 시 무공 정수 50% 획득
- 세이브 슬롯 삭제 가능

### 4.13 UI/UX 요구사항

**FR-040**: 전투 UI
전투 화면은 다음 요소를 포함:
- **상단**: 적 정보 (체력, 의도, 버프/디버프)
- **중앙**: 전투 필드 (적 캐릭터, 플레이어 캐릭터)
- **하단**:
  - 손패 (카드 5-10장)
  - 덱/버리기/소진 더미 카운터
  - 현재/최대 내공 표시
  - 턴 종료 버튼
- **좌측**: 플레이어 체력, 방어도, 버프/디버프
- **우측**: 뽑기 더미, 버리기 더미, 소진 더미 아이콘

**FR-041**: 맵 UI
맵 화면은 다음 요소를 포함:
- 노드 네트워크 시각화
- 현재 위치 하이라이트
- 노드 타입 아이콘 표시
- 상단: 플레이어 상태 (체력, 골드, 층수)
- 우측: 덱 정보, 유물 목록 버튼

**FR-042**: 카드 상세 정보
- 카드에 마우스 오버 시 확대 및 상세 정보 표시
- 표시 정보:
  - 카드 이름
  - 비용
  - 타입 및 계열
  - 효과 설명
  - 업그레이드 여부
  - 업그레이드 시 변화 미리보기 (업그레이드 화면에서)

**FR-043**: 접근성
- 모든 카드/유물/버프는 툴팁으로 설명 제공
- 키보드 단축키 지원:
  - E: 턴 종료
  - Q: 물약 사용
  - 1-0: 카드 선택
  - Tab: 적 순환
  - Esc: 메뉴 열기
- 색맹 모드 옵션
- 텍스트 크기 조절 옵션

### 4.14 사운드 및 음악

**FR-044**: 배경 음악
- 각 지역별 고유한 배경음악
- 전투 시 전투 테마 전환
- 보스 전투 시 전용 보스 음악
- 메인 메뉴 음악

**FR-045**: 효과음
필수 효과음:
- 카드 사용 효과음 (타입별 차별화)
- 피해 입힘/받음 효과음
- 버프/디버프 획득 효과음
- 카드 드로우 효과음
- 골드 획득 효과음
- 유물 획득 효과음
- 경지 돌파 효과음
- 버튼 클릭 효과음

### 4.15 튜토리얼

**FR-046**: 초회 튜토리얼
첫 게임 시작 시 튜토리얼 제공:
1. 기본 조작 설명 (카드 드래그, 턴 종료)
2. 내공 시스템 설명
3. 적 의도 읽기
4. 첫 전투 (약한 적)
5. 카드 보상 선택
6. 맵 이동 설명

**FR-047**: 컨텍스트 도움말
- 처음 접하는 요소 발견 시 간단한 팝업 설명
- 예시: 처음 이벤트 노드 방문 시 설명
- 설정에서 도움말 끄기 가능

### 4.16 설정

**FR-048**: 게임 설정
설정 메뉴에서 조정 가능:
- **그래픽**:
  - 해상도 (창 모드/전체화면)
  - 화면 효과 ON/OFF
  - VSync ON/OFF
- **오디오**:
  - 마스터 볼륨
  - 음악 볼륨
  - 효과음 볼륨
- **게임플레이**:
  - 빠른 전투 모드 (애니메이션 단축)
  - 카드 드래그 확인
  - 턴 종료 확인
  - 색맹 모드
- **언어**: 한국어 (추후 영어, 중국어 등 확장 가능)

## 5. 비기능 요구사항 (Non-Goals / Out of Scope)

본 PRD 범위에서 **제외**되는 항목:

**NG-001**: 멀티플레이어 기능
- 본 게임은 싱글플레이어 전용입니다
- PvP, 협동 모드는 포함하지 않습니다

**NG-002**: 실시간 서버 연동
- 온라인 리더보드는 초기 버전에 포함하지 않습니다
- 클라우드 세이브는 추후 업데이트로 고려

**NG-003**: 커스텀 덱 빌더 (사전 덱 구성)
- 게임 시작 전 덱 구성 기능은 제공하지 않습니다
- 덱은 런 중에만 구축됩니다

**NG-004**: 캐릭터 커스터마이징
- 외형 커스터마이징은 제공하지 않습니다
- 캐릭터는 선택한 분파에 따라 고정됩니다

**NG-005**: 3D 그래픽
- 본 게임은 2D 기반입니다
- 3D 요소는 포함하지 않습니다

**NG-006**: 음성 더빙
- 텍스트 기반 스토리텔링만 제공합니다
- 음성 더빙은 포함하지 않습니다

**NG-007**: 실시간 시간 제한
- 턴 제한 시간은 없습니다
- 플레이어는 원하는 만큼 고민할 수 있습니다

**NG-008**: 모바일 플랫폼
- 초기 출시는 PC만 지원합니다
- 모바일은 추후 검토 대상입니다

## 6. 디자인 고려사항 (Design Considerations)

### 6.1 아트 스타일
- **전체 톤**: 수묵화 + 현대적 일러스트 혼합
- **카드 일러스트**: 각 무공의 특성을 표현하는 동적인 자세
- **캐릭터**: 2D 스프라이트, 기본 애니메이션 (대기, 공격, 피격, 사망)
- **배경**: 강호를 연상시키는 중국풍 풍경 (산, 사막, 설원 등)
- **UI**: 한지 질감과 먹선을 활용한 한국/중국 전통 느낌

### 6.2 폰트
- **메인 폰트**: 고딕 계열 (가독성 우선)
- **강조 폰트**: 서예체 (제목, 카드 이름 등)
- **크기**: 본문 16-18pt, 카드 텍스트 14pt, 설명 12pt

### 6.3 색상 팔레트
- **배경**: 베이지, 갈색 계열
- **강조**: 금색, 붉은색
- **등급 구분**:
  - 평범: 회색 #808080
  - 고급: 파란색 #4A90E2
  - 진귀: 보라색 #9B59B6
  - 전설: 금색 #F39C12

### 6.4 애니메이션
- **카드 사용**: 카드가 확대되며 빛나는 효과
- **피해**: 적이 붉게 깜빡이며 흔들림
- **방어도**: 방패 아이콘이 캐릭터 앞에 나타남
- **버프/디버프**: 아이콘이 떠오르며 표시
- **카드 드로우**: 덱에서 손으로 카드가 날아옴

### 6.5 참고 UI/UX
- **맵 시스템**: Slay the Spire 스타일 노드 맵
- **전투 화면**: Monster Train, Griftlands 레이아웃 참고
- **카드 디자인**: 무협 특유의 붓글씨 느낌 + 명확한 아이콘

## 7. 기술 고려사항 (Technical Considerations)

### 7.1 기술 스택
- **게임 엔진**: Unity (C#) 또는 Godot (GDScript/C#)
  - 권장: Unity (더 많은 레퍼런스와 에셋)
- **버전 관리**: Git
- **아트 툴**: Adobe Photoshop, Clip Studio Paint
- **사운드 툴**: Audacity, FL Studio

### 7.2 성능 요구사항
- **최소 사양**:
  - OS: Windows 7/Mac OS X 10.12
  - CPU: 2.0 GHz Dual Core
  - RAM: 2 GB
  - GPU: Intel HD Graphics 4000
  - 저장공간: 500 MB
- **타겟 프레임레이트**: 60 FPS
- **로딩 시간**: 씬 전환 시 2초 이내

### 7.3 데이터 구조
**카드 데이터**:
```json
{
  "id": "card_001",
  "name": "일검",
  "cost": 1,
  "type": "attack",
  "category": "sword",
  "rarity": "common",
  "baseDamage": 6,
  "upgradedDamage": 9,
  "description": "적에게 {damage} 피해",
  "keywords": []
}
```

**유물 데이터**:
```json
{
  "id": "relic_001",
  "name": "낡은 검집",
  "rarity": "common",
  "description": "전투 시작 시 방어도 3 획득",
  "effectType": "combat_start",
  "effectValue": 3
}
```

**적 데이터**:
```json
{
  "id": "enemy_001",
  "name": "산적",
  "maxHealth": 40,
  "actions": [
    {"type": "attack", "damage": 6, "weight": 60},
    {"type": "defend", "block": 8, "weight": 40}
  ],
  "goldReward": [20, 30]
}
```

### 7.4 절차적 생성 알고리즘
- **맵 생성**: 레이어별 노드 배치, 보장된 경로 생성
- **카드 보상**: 가중치 기반 랜덤 (등급, 계열 고려)
- **이벤트 선택**: 지역별 이벤트 풀, 조건부 필터링
- **적 배치**: 층수에 따른 난이도 스케일링

### 7.5 세이브 파일 구조
- **형식**: JSON (읽기 쉽고 디버깅 용이)
- **암호화**: 체크섬을 통한 무결성 검증 (치트 방지)
- **저장 경로**:
  - Windows: `%APPDATA%/GanghoGame/saves/`
  - Mac: `~/Library/Application Support/GanghoGame/saves/`
  - Linux: `~/.local/share/GanghoGame/saves/`

### 7.6 모듈화 구조
게임은 다음 모듈로 구성:
1. **Core**: 게임 루프, 상태 관리
2. **Combat**: 전투 로직, 카드 시스템
3. **Map**: 맵 생성, 노드 관리
4. **Data**: 카드/유물/적 데이터 관리
5. **UI**: 모든 UI 컴포넌트
6. **Audio**: 사운드 매니저
7. **Save**: 세이브/로드 시스템
8. **Meta**: 메타 진행, 업적

### 7.7 확장성 고려
- 카드/유물/적은 데이터 기반으로 쉽게 추가 가능하도록 설계
- 새로운 분파 추가를 위한 템플릿 제공
- 모딩 지원 고려 (JSON 파일 수정 허용)
