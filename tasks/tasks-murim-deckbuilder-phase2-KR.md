# 작업 목록: 강호무적 - Phase 2 수직 슬라이스

## 프로젝트 개요

이 작업 목록은 무협 덱 빌딩 로그라이크 게임 "강호무적"의 **Phase 2: 수직 슬라이스** 개발을 다룹니다. 목표는 2-3개월 내에 완전한 1개 지역 플레이 경험을 구현하는 것입니다.

**Phase 2 성공 기준:**
- 노드 기반 맵 시스템 구현 (12-18개 노드)
- 카드 50장으로 확장 (다양한 계열)
- 유물 시스템 (20개 유물)
- 상점, 휴식, 이벤트 노드 기능
- 보스 전투 (고유 패턴)
- 메타 진행 시스템 (무공 정수, 5개 영구 업그레이드)
- 1개 지역 완주 가능
- 외부 테스터 긍정 반응

**전제 조건:** Phase 1 프로토타입 완료

**참고:** 상세 요구사항은 `prd-murim-deckbuilder-roguelike-KR.md`를 참조하세요.
